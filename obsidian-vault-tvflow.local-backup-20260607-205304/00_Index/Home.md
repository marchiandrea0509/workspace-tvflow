## Governance

Raw evidence stays in project reports/exports. This vault stores distilled knowledge, links, decisions, and repeated patterns.

Use labels:

- `observation` — one case, not a rule
- `pattern` — repeated enough to watch
- `candidate_rule` — proposed workflow change
- `approved_rule` — implemented in prompts/tools/docs
- `safety_critical` — patch immediately if confirmed
