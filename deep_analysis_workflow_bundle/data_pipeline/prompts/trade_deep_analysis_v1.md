# Trade Deep Analysis v1

Use this whenever Gray gives a "deep analysis" on a symbol/setup.

## Mission
Do not drift into purely qualitative commentary.
A deep analysis must contain a **quantitative trade spine**.

## Principle
**PRICE_FIRST_CONTEXT_SECOND**
- price/features decide the setup
- context adjusts conviction, urgency, and posture
- context does not replace execution structure

## Mandatory sections
### 1) Core read
Always include:
- bias
- setup
- conviction
- rank / board status if known
- time horizon

### 2) Quantitative levels
Always include labeled levels/zones:
- current/recent price area
- tactical support / resistance
- higher-timeframe support / resistance when available

### 3) Setup quality
Always say whether this is:
- a good market and a good entry
- a good market but bad entry
- or simply not a good trade

### 4) Execution plans
Give at least:
- preferred setup
- entry zone
- stop / invalidation
- target ladder
- risk/reward comment
- optional secondary setup if valid

### 5) Orderability decision
This is mandatory.
You must answer clearly:
- `placeable_now`
- `placeable_conditional_only`
- `not_placeable_yet`

Then explain in plain English:
- **market order now?** yes/no
- **ladder limit orders?** yes/no, and where
- **stop entry orders?** yes/no, and where
- **why?**

## Hard rule
If the trade is not ready, still say whether **conditional resting orders** can be placed.
Do not leave the user unsure whether they can queue:
- a ladder limit order
- a stop-entry order
- or neither

## Good wording examples
- "Not a good market order now, but conditional ladder limits are valid in 35.20-35.90 if the weak rebound scenario remains intact."
- "Not placeable yet: no market order, no ladder limits, and no stop-entry order until price either reclaims structure or breaks support with acceptance."
- "Placeable conditional only: sell-stop below 34.64 is valid, but only if acceptance logic is part of the plan."

## Final Gray call
End with a plain-language trader verdict such as:
- "Good market, bad entry"
- "Conditional short only"
- "No order yet"
- "Ready to place ladder"

Keep it direct and practical.
